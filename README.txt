This is the Drupal 8 development version of FillPDF. Only use this if you are
using Drupal 8. Otherwise, use the code at https://drupal.org/project/fillpdf.
This repository has its own issue and pull request queue. Please only use that
for Drupal 8-related stuff. Otherwise, file an issue at https://drupal.org/project/issues/fillpdf.

The setup for this module is essentially the same regardless of Drupal version.

See the full documentation on drupal.org: http://drupal.org/documentation/modules/fillpdf
